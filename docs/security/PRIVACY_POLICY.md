## 🎯 概述

### 适用范围

本隐私政策适用于：
- sfsEdgeStore 软件
- 我们的网站
- 相关的服务和支持

### 我们的承诺

我们承诺：
- 尊重您的隐私权
- 透明的数据处理
- 安全的数据保护
- 提供数据控制权

### 法律依据

我们遵守以下隐私法律法规：
- 《中华人民共和国个人信息保护法》
- 《通用数据保护条例》(GDPR)（如适用）
- 其他适用的隐私法律

## 💾 我们收集的数据

### 客户数据

**我们不访问、不存储、不处理您的业务数据**：
- 设备传感器数据
- EdgeX Foundry 事件数据
- 存储在 sfsDb 中的所有数据
- 您的业务敏感数据

这些数据完全在您的控制之下，存储在您的设备或服务器上。

### 我们可能收集的信息

#### 1. 账户信息

如果您使用我们的付费服务，我们可能收集：
- 公司名称
- 联系人姓名
- 联系邮箱
- 联系电话
- 账单地址
- 付款信息（由支付服务商处理）

#### 2. 技术信息

为提供支持和改进服务，我们可能收集：
- 软件版本
- 操作系统版本
- 错误日志（您主动提交）
- 性能统计（您主动启用）
- 配置信息（不含敏感数据）

#### 3. 通信信息

当您联系我们时，我们可能收集：
- 您的联系方式
- 通信内容
- 支持工单历史

## 🔧 数据使用方式

### 使用目的

我们使用收集的信息用于：

1. **提供服务**
   - 处理您的订单
   - 提供技术支持
   - 发送服务通知

2. **改进服务**
   - 分析使用情况
   - 改进软件功能
   - 优化用户体验

3. **安全保护**
   - 防止欺诈
   - 保护账户安全
   - 遵守法律义务

4. **通信**
   - 发送重要更新
   - 提供安全公告
   - 营销信息（仅在您同意时）

### 不使用方式

我们不会：
- 访问您的业务数据
- 将您的数据用于广告
- 出售您的个人信息
- 未经同意共享您的数据

### 我们共享的情况

我们仅在以下情况下共享信息：

1. **服务提供商**

2. **法律要求**
   - 遵守法院命令
   - 响应政府要求

3. **您的同意**
   - 您明确同意
   - 您主动选择

我们使用以下第三方服务：

| 服务类型 | 服务商 | 用途 |
|---------|--------|------|

| 客户支持 | [支持系统] | 工单管理 |

## 🔒 数据存储与保护

### 数据存储位置

- **您的业务数据**：存储在您的设备/服务器上
- **账户信息**：存储在我们的安全服务器上
- **支持数据**：存储在我们的支持系统中

### 数据保留期限

| 数据类型 | 保留期限 |
|---------|---------|
| 账户信息 | 账户活跃期间 + 7 年 |
| 支持记录 | 工单关闭后 + 3 年 |
| 支付记录 | 法律要求的期限 |
| 通信记录 | 通信后 + 3 年 |

### 安全措施

我们采取以下安全措施：
- 加密传输（TLS）
- 加密存储（AES-256）
- 访问控制和认证
- 定期安全审计
- 员工隐私培训

## 👤 用户权利

### 您的权利

根据适用的隐私法律，您可能拥有以下权利：

| 权利 | 说明 |
|------|------|
| **访问权** | 请求访问我们持有的您的个人信息 |
| **更正权** | 请求更正不准确的个人信息 |
| **删除权** | 请求删除您的个人信息 |
| **限制处理权** | 请求限制处理您的个人信息 |
| **数据可携带权** | 请求获取您的数据的机器可读副本 |
| **反对权** | 反对处理您的个人信息 |

### 如何行使权利

**方式**：
- **邮件**：privacy@[您的域名].com
- **在线表单**：[隐私请求表单链接]

**所需信息**：
- 您的身份验证信息
- 具体的请求内容
- 联系方式

**响应时间**：
- 确认收到：24 小时内
- 完整响应：30 天内

## 👶 儿童隐私

### 儿童数据

我们的服务不面向 13 岁以下的儿童。

如果您认为我们收集了儿童的个人信息，请立即联系我们，我们将：
- 立即删除相关信息
- 调查并解决问题

## 执行摘要（主打线上销售）

sfsEdgeStore 是一款轻量级边缘计算数据存储适配器，专为物联网边缘场景设计。作为 EdgeX Foundry 和 sfsDb 之间的桥梁，它提供高效的本地数据读写和缓存能力。

**核心销售策略：主打线上渠道（60% 收入）**，辅以渠道伙伴（25%）和战略直销（15%），实现快速规模化增长。

### 核心价值

- **轻量高效**：内存 < 50MB，CPU < 5%
- **边缘友好**：可独立于中心系统运行，网络中断不影响本地数据采集
- **易于集成**：与 EdgeX Foundry 原生集成，支持 MQTT 和 HTTP API
- **商业级功能**：认证授权、监控告警、数据备份、升级支持

### 2026 年销售目标（线上为主）

- **签约客户数**：75 家（50 家在线 + 17 家渠道 + 8 家直销）
- **年度营收**：500 万元（300 万在线 + 145 万渠道 + 55 万直销）
- **在线收入占比**：60%
- **试用注册数**：2,000 人
- **试用→付费转化率**：13%
- **市场渗透率**：在 EdgeX Foundry 生态中达到 5%
